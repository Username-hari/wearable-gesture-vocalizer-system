# Audio Tracks Reference

Copy the following MP3 files to the **root directory** of a FAT32-formatted microSD card.

| File Name | Spoken Phrase | Gesture |
|-----------|--------------|---------|
| `001.mp3` | "Hello" | Open palm |
| `002.mp3` | "Thank you" | Thumb up |
| `003.mp3` | "Yes" | Closed fist |
| `004.mp3` | "No" | Index point |
| `005.mp3` | "Help me" | Peace sign |
| `006.mp3` | "Water, please" | W shape |

## Recording Tips

- Use a clear, neutral voice at a consistent volume
- Record at 44.1 kHz, 16-bit mono for best DFPlayer compatibility
- Keep each clip under 3 seconds
- Use tools like Audacity (free) or any TTS engine to generate files
